SYNTAX

JSON syntax allows the representation of three type of values:

  1. Simple values - Strings, numbers, Booleans, and null can all be represented in JSON using the same syntax as JavaScript. The special value undefined is not supported.
  2. Objects - The first complex data type, objects represent ordered key-value pairs. Each value may be a primitive type or a complex type.
  3. Arrays - The second complex data type, arrays represent an ordered list of values that are accessible via a number index. The values may be of any type, including simple values, objects, and even other arrays.

Simple Values

In its simplest form, JSON represents a small number of simple values. For examples, the following is valid JSON: {% highlight js %} 5 {% endhighlight %} This is JSON that represents the number 5.
